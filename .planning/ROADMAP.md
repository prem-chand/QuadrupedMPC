# Roadmap: QuadrupedMPC

## Project Overview

MIT Cheetah-style convex MPC controller for Unitree Go2 quadruped robot.

## Milestones

- [v1.0](./milestones/v1.0-ROADMAP.md) — Core MPC-WBC stack (completed)
- [v1.1](./milestones/v1.1-ROADMAP.md) — MIT Cheetah parity (completed)

## Current Status

**v1.1 Complete** — MIT Cheetah parity features:
- State estimation (Kalman filter)
- Controller frequency (100 Hz MPC / 500 Hz WBC)
- Balance controller for push recovery
